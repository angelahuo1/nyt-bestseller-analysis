# Project 2 – Exploratory Analysis & NLP

Uses the cleaned dataset to perform exploratory analysis and natural language processing on bestseller descriptions.

## Notebooks
### `exploratory_da.ipynb`
- Category and author trends
- Rank distributions
- Weeks-on-list analysis
- Publisher patterns
- Visualizations

### `nlp_analysis.ipynb`
- Text cleaning + lemmatization
- Word frequency + wordcloud
- TF–IDF features
- Topic modeling (LDA)
- Sentiment analysis (VADER)

## Data
Input:
- `data/analysis/bestsellers_analysis_ready.csv`

Output:
- `top_tfidf_terms.csv`
- `topic_summary.csv`
- `bestsellers_nlp_topics_sentiment.csv`
